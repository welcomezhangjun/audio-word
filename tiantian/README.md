# 六臂哪吒贪吃蛇游戏

这是一个以中国神话人物六臂哪吒为主题的贪吃蛇游戏。游戏采用经典的贪吃蛇玩法，但增加了六臂哪吒的特色元素，包括独特的穿墙能力，让游戏更具挑战性和趣味性。

## 游戏特色

- **六臂哪吒主题**: 蛇头为六臂哪吒形象，食物为火尖枪
- **穿墙能力**: 每获得50分可以穿墙一次，让你的蛇能够穿越边界
- **渐进难度**: 随着得分增加，游戏速度会逐渐加快，增加挑战性
- **视觉效果**: 精美的SVG图像和动态效果，提升游戏体验
- **适合初学者**: 简单的控制方式，适合各年龄段玩家
- **宽敞游戏区域**: 550x550像素的游戏区域，提供更大的活动空间
- **移动端支持**: 响应式设计和触摸控制，支持在手机和平板上游玩

## 如何开始游戏

1. 打开`index.html`文件启动游戏
2. **桌面设备**：使用键盘方向键控制六臂哪吒蛇的移动方向
3. **移动设备**：使用屏幕上的方向按钮控制蛇的移动
4. 吃到火尖枪食物可以增加分数和蛇的长度
5. 每获得50分会获得一次穿墙能力，可以穿越游戏边界
6. 当穿墙能力可用时，蛇头周围会出现橙色光环提示

## 游戏规则

- 使用方向键或触摸按钮控制蛇的移动
- 吃到食物可以增加10分
- 撞到自己的身体会导致游戏结束
- 撞到墙壁会导致游戏结束，除非有穿墙能力
- 每获得50分可以获得一次穿墙能力
- 穿墙能力使用后需要再次积累50分才能获得

## 游戏设置

- **桌面设备**：游戏画布为550 x 550像素
- **移动设备**：游戏画布自适应屏幕宽度，保持正方形比例
- 网格大小：30像素
- 初始速度：700毫秒/步
- 速度增加：每吃一个食物后速度增加2毫秒
- 最小速度：150毫秒/步（最快）

## 设备兼容性

- **桌面设备**：支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
- **移动设备**：支持iOS和Android设备上的主流浏览器
- **响应式设计**：自动适应不同屏幕尺寸，优化移动体验
- **触摸控制**：针对触摸屏设备优化的控制按钮

## 文件结构

- `index.html`: 游戏主页面
- `style.css`: 游戏样式表
- `script.js`: 游戏核心逻辑
- `assets/`: 游戏资源目录
  - `nezha-six-arms.svg`: 六臂哪吒头像
  - `food.svg`: 火尖枪食物图像

## 测试用例

1. **基本移动**: 使用方向键或触摸按钮控制蛇的移动，确保蛇能够正确响应输入
2. **食物生成**: 确保食物不会生成在蛇身上，且能被正确吃到
3. **碰撞检测**: 测试蛇撞到自己身体时游戏结束
4. **穿墙能力**: 测试获得50分后穿墙能力是否可用，使用后是否能穿越边界
5. **分数计算**: 确保吃到食物后分数正确增加，且穿墙能力在正确的分数点激活
6. **游戏结束**: 测试游戏结束后显示分数并可以重新开始游戏
7. **速度增加**: 确保随着得分增加，游戏速度逐渐加快
8. **响应式布局**: 测试在不同设备和屏幕尺寸下的显示效果
9. **触摸控制**: 验证移动设备上的触摸控制是否灵敏准确
