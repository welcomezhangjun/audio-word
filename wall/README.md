# 金句墙 - Wisdom Wall

## 项目概述
一个展示教育理念和人生感悟的现代化金句墙，采用苹果风格的简洁设计。

## 设计理念
- **简洁至上**：遵循苹果的设计哲学，去除不必要的装饰
- **内容为王**：突出金句内容，让智慧的话语成为焦点
- **交互友好**：提供直观的浏览和搜索体验
- **响应式设计**：适配不同设备和屏幕尺寸

## 功能特性
- 金句展示墙    
- 按作者分类
- 关键词搜索
- 收藏功能
- 响应式布局

## 技术栈
- HTML5 + CSS3
- JavaScript (ES6+)
- SVG 图标设计

## 项目目标
创建一个优雅、实用的平台来展示和分享有价值的教育理念与人生感悟。
