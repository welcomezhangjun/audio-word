# 语音转文字工具

这是一个简单易用的实时语音转文字工具，具有以下特点：

## 功能特性
- 实时语音识别
- 自动添加标点符号
- 文字复制功能
- 简洁美观的界面设计

## 使用说明
1. 点击"开始录音"按钮开始语音识别
2. 说话时文字会实时显示在下方文本框中
3. 点击"停止录音"按钮结束录音
4. 使用"复制文字"按钮可以复制所有转换的文字

## 注意事项
- 需要使用支持Web Speech API的现代浏览器（推荐使用Chrome）
- 首次使用时需要允许浏览器使用麦克风
- 请在安静的环境下使用，以获得最佳识别效果
